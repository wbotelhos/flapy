Default configuration:

This example shows how to use the default configuration with div boxes.

Box 1
Box 2
Box 3
$('#tab').flapy({
  headers: [{ text: 'Ruby' }, { text: 'Python' }, { text: 'Java' }]
});

<div id="tab">
  <div>Box 1</div>
  <div>Box 2</div>
  <div>Box 3</div>
</div>

Including content tag into the header:

You can choose some HTML tag to put inside the li header:

Box 1
Box 2
Box 3
$('selector').flapy({
  headers: [{ tag: 'h3', text: 'Ruby' }, { tag: 'h3', text: 'Python' }]
});

Do not forget the style: .flapy-header li h2 and .flapy-header li.active h2.

Activating a tab:

By default the first tab is actived, but you can choose which tab will start activated with the active option:

Box 1
Box 2
Box 3
$('selector').flapy({
  headers: [{ text: 'Ruby' }, { text: 'Python', active: true }]
});

Using a custom wrapper:

Normally we have a header and several boxes that hide and show up. When we want to add some fixed element outside the boxes, we need to add a wrapper:

Fixed element

Box 1
Box 2
Box 3
$('selector').flapy({
  headers : [{ text: 'Ruby' }, { text: 'Python' }]
  wrapper: 'div'
});

<div id="default">
  <div>
    <p>Outside the box</p>
    <div>Box 1</div>
    <div>Box 1</div>
    <div>Box 1</div>
  </div>
</div>

Styling the tabs:

You can disable the auto borders to design your own tab style.

Box 1
Box 2
Box 3
$('selector').flapy({
  border : true,
  headers: [{ tag: 'h3', text: 'Ruby' }, { tag: 'h3', text: 'Python' }]
});

Changing the settings globally:

You can change any option mentioning the scope $.fn.flapy.defaults. + option_name. It must be called before you bind the plugin. 

$.fn.flapy.defaults.box = 'div';
$.fn.flapy.defaults.wrapper = 'p';

Options:

border: true

Changes the tab borders automatic.

box: 'div'

The children that will represent the boxes.

headers: []

Array of object containing tag, text and active attributes representing each tab.

wrapper: undefined

An element that will keeps all boxes.
